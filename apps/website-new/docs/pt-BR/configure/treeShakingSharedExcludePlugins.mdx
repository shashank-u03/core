# treeShakingSharedExcludePlugins

Configures o plugin names para exclude during o dependência compartilhada tree shaking/fallback process. Esses plugins vai ser ignored e vai não participate no dependência compartilhada tree shaking/fallback process.

- Tipo: `string[]`
- Obrigatório: No
- Padrão: `['HtmlWebpackPlugin','HtmlRspackPlugin']`
