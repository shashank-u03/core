# treeShakingDir

Quando o [dependência compartilhada Tree Shaking feature](/guide/advanced/shared-tree-shaking.html) é habilitado, {props.brandName || props.name || 'Module Federation'} vai extract o exporta de unused módulos shared. Este opção specifies o output diretório for those fallback assets.

- Tipo: `string`
- Obrigatório: No
- Padrão: `undefined`
