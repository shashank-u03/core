# Presets

## Shared

Por padrão, certain Next.js internals são automatically shared para optimize your aplicação.
It's important para note esse manually compartilhamento these internals pode lead para erros, as they são já handled for você.

- `next/dynamic`
- `next/head`
- `next/link`
- `next/router`
- `next/script`
- `react`
- `react/`
- `react-dom`
- `react-dom/`
- `styled-jsx`
- `styled-jsx/style`

## Exponha Pages

Você pode automatically exponha all pages via o plugin

```javascript
new NextFederationPlugin({
  name: 'remoteApp',
  filename: '',
  remotes: {},
  exposes: {},
  shared: {},
  extraOptions: {
    exposePages: true
  },
});
```

Este vai make pages disponível para importe like:

- `import Page from 'remoteApp/pages/index'`
- `import Page from 'remoteApp/pages/shop/product'`

## Build Opções
### Public Caminho

Public Caminho é definir para "auto"

`config.output.publicPath: "auto"`

### Target

Webpack output target é definir para `async-node` de `node`

`config.target = "async-node"`
