# React Practice

React apps não usam one fixed React-specific Module Federation plugin. Choose o integração based no bundler usado by seu projeto primeiro, then use these React scenario guias.

| Project tipo | Read next |
| --- | --- |
| Rsbuild | [Rsbuild](/integrations/build-tool/rsbuild.html) |
| Rslib | [Rslib](/integrations/build-tool/rslib.html) |
| Rspack | [Rspack](/integrations/bundler/rspack.html) |
| Webpack | [Webpack](/integrations/bundler/webpack.html) |
| Vite | [Vite](/integrations/build-tool/vite.html) |
| Runtime remote carregamento | [Runtime Installation](/guide/runtime/index.html#installation) |

Se você precisa carregue remote React componentes, continue com [Bridge](/guide/bridge/overview.html). Se remote componentes need dados fetching, dados cache, ou pré-busca, continue com [Data Management](/guide/data/data-fetch.html).
