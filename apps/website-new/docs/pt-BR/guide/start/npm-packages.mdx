# Pacotes NPM

Este documento showcases all npm pacote information maintained pelo Module Federation team.

## @module-federation/enhanced

![](https://img.shields.io/npm/v/@module-federation/enhanced?style=flat-square&colorA=564341&colorB=EDED91)

O pacote principal do Module Federation, atuando como um plugin de build para Webpack, Rspack plugin de build, e dependência de entrada do runtime.

- [npm](https://npmjs.com/package/@module-federation/enhanced)
- [Source Code](https://github.com/module-federation/core/tree/main/packages/enhanced)
- [Runtime Documentation](/guide/runtime/index.html)
- [Rspack Build Plugin](/integrations/bundler/rspack.html)
- [Webpack Build Plugin](/integrations/bundler/webpack.html)

## @module-federation/runtime

![](https://img.shields.io/npm/v/@module-federation/runtime?style=flat-square&colorA=564341&colorB=EDED91)

O pacote de runtime do Module Federation, geralmente usado com @module-federation/enhanced para aproveitar os recursos do Runtime, e pode ser instalado separadamente se no ferramenta de build é obrigatória.

- [npm](https://npmjs.com/package/@module-federation/runtime)
- [Source Code](https://github.com/module-federation/core/tree/main/packages/runtime)
- [Documentation](/guide/runtime/index.html)

## @module-federation/rspack

![](https://img.shields.io/npm/v/@module-federation/rspack?style=flat-square&colorA=564341&colorB=EDED91)

O plugin de build do Rspack para Module Federation, geralmente usado com @module-federation/enhanced, e não pode ser usado independentemente sem o API de runtime.

- [npm](https://npmjs.com/package/@module-federation/rspack)
