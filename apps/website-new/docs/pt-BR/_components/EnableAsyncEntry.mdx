Se o `@module-federation/modern-js-v3` plugin é usado, o corresponding asynchronous entry vai ser habilitado based no builder tipo por padrão.

But se it é [build mode](https://modernjs.dev/en/guides/concept/entries.html#build-mode-entry), then você still precisa definir o asynchronous entry manually.

Next, nós vai demonstrate como para habilite asynchronous entry.

Crie o bootstrap.js arquivo e copy o contents do original arquivo de entrada here:

```diff title="bootstrap.js"
+ import React from 'react';
+ import ReactDOM from 'react-dom';
+ import App from './App';
+ ReactDOM.render(<App />, document.getElementById('root'));
```

Modify o content do original arquivo de entrada e reference bootstrap.js instead:

```diff title="index.js"
+ import('./bootstrap');
- import React from 'react';
- import ReactDOM from 'react-dom';
- import App from './App';
- ReactDOM.render(<App />, document.getElementById('root'));
```
